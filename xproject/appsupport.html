This is a developer part-time project management app where developers can create part-time projects that manage the payment of part-time projects or view reports.

1. Project management:

The amount, profile, project name, and project type of the project can be managed;

2. Payment management:

After selecting a specific item, you can view the item's payment list and add payment information: payment amount, payment time, payment note;

3. Report function:

View the income of the project through the pie chart, you can view it through the time horizon of this week, this month, and this year, or you can manually select the time span.
